# Ping Platform CI/CD
 
Automated deployment of PingFederate and PingDirectory to Azure Kubernetes Service
using Azure DevOps Pipelines, Docker, and Helm.
 
## Services
- **PingFederate** — Identity federation / SSO (namespace: `ping-federate`)
- **PingDirectory** — LDAP directory (namespace: `ping-directory`)
